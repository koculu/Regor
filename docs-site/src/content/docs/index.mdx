Regor empowers developers to craft high-performance, reactive user interfaces effortlessly. Whether you're building a single-page app, desktop GUI, or hybrid solution, Regor’s intuitive API and uncompromising performance let you focus on what matters: your users.

<CardGrid stagger>
  <Card title="Vue-Like Syntax" icon="seti:vue">
    Template conventions familiar to Vue.js developers for an instant productivity boost.
  </Card>

  <Card title="Native TypeScript" icon="seti:typescript">
    Full type-safety with zero workarounds—define components and props in TS.
  </Card>

  <Card title="No Build Step" icon="rocket">
    Run directly in the browser or Node.js—no bundler, no compilation delays.
  </Card>

  <Card title="Secure VM" icon="seti:lock">
    Built-in JavaScript VM enforces CSP-friendly, sandboxed template evaluation.
  </Card>

  <Card title="Flexible Reactivity" icon="random">
    Granular state control via ref, computed, watchEffect, and custom hooks.
  </Card>

  <Card title="Partial rendering" icon="laptop">
    Partially render any part in the raw html.
  </Card>

  <Card title="Reentrance" icon="seti:json">
    Mount multiple app contexts in the same DOM tree—dynamic components on the fly.
  </Card>

  <Card title="Library-Friendly" icon="puzzle">
    Seamlessly coexists with other DOM-manipulating libraries and micro-frontends.
  </Card>
</CardGrid>
